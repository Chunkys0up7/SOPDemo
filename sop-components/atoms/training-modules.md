# Required Security Training Modules

## Core Modules (All Employees)

1. **Information Security Basics** (30 minutes)
   - Password security
   - Phishing awareness
   - Device security

2. **Data Protection & Privacy** (45 minutes)
   - GDPR compliance
   - Data classification
   - Handling sensitive information

3. **Social Engineering Awareness** (20 minutes)
   - Common attack vectors
   - Reporting suspicious activity

4. **Incident Response** (15 minutes)
   - What to do if you suspect a breach
   - Contact information for security team

## Specialized Modules (Role-Based)

### For Developers
- Secure coding practices
- API security
- Secret management

### For Management
- Security leadership
- Risk assessment
- Vendor security evaluation

## Completion Requirements

- All modules must be completed within **first week** of employment
- Passing score: **80%** or higher
- Annual refresher training required
- Certificates will be issued upon completion
