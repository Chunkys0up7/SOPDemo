# Complete HR Onboarding Workflow

## Overview

This comprehensive workflow covers all HR-related onboarding activities for new employees, from paperwork to benefits enrollment to compliance training.

## Pre-Start (Week Before Start Date)

### HR Preparation Checklist

- [ ] Offer letter signed and returned
- [ ] Background check completed
- [ ] Reference checks completed
- [ ] I-9 documentation requested
- [ ] Welcome email sent with first-day details
- [ ] Employee file created in HRIS
- [ ] Badge requested from Security
- [ ] IT notified of start date

---

## Day 1: Welcome and Orientation (3-4 hours)

### Morning Session (9:00 AM - 12:00 PM)

**9:00 - 9:30: Welcome and Paperwork**

HR reviews and collects:

- [ ] I-9 documentation (original documents)
- [ ] W-4 federal tax withholding
- [ ] State tax withholding forms
- [ ] Direct deposit authorization
- [ ] Emergency contact information

{{include: emergency-contact}}

**9:30 - 10:00: Legal Agreements**

Employee reviews and signs:

- [ ] Employee handbook acknowledgment
- [ ] Confidentiality agreement
- [ ] Code of conduct
- [ ] At-will employment acknowledgment
- [ ] Background check authorization (if not already done)

#### Confidentiality Agreement

{{include: confidentiality-agreement}}

**10:00 - 10:15: Break**

**10:15 - 11:00: Company Overview**

Presentation covering:

- Company history and mission
- Organizational structure
- Products and services
- Client/customer base
- Recent achievements and news
- Strategic goals

**11:00 - 11:45: Benefits Overview**

{{include: benefits-overview}}

**11:45 - 12:00: Benefits Q&A**

### Lunch (12:00 - 1:00 PM)

- Hosted lunch with team
- Informal getting-to-know-you time
- Manager or team lead attends

### Afternoon Session (1:00 - 5:00 PM)

**1:00 - 2:00: Workplace Policies**

Review of:

- Working hours and attendance
- PTO request procedures
- Sick leave policies
- Remote work guidelines
- Communication expectations
- Dress code
- Use of company resources

**2:00 - 3:00: HRIS and Payroll Systems**

Hands-on training:

- [ ] Login to employee self-service portal
- [ ] Review personal information
- [ ] Set up direct deposit
- [ ] Learn how to request time off
- [ ] Submit timesheets (if applicable)
- [ ] Access pay stubs
- [ ] Update tax withholdings

**3:00 - 3:15: Break**

**3:15 - 4:30: Start Compliance Training**

{{include: compliance-training}}

**Note**: Complete remaining training by end of Week 1

**4:30 - 5:00: Day 1 Wrap-up**

- Review completed checklist
- Address any questions
- Provide access credentials
- Schedule follow-up meetings

### Days 2-5: Benefits Enrollment

{{include: benefits-enrollment}}

**Daily Goals**:

Continue working on compliance training modules:

- [ ] Module 1: Code of Conduct

### End of Week Meeting (Friday, Week 1)

**30-minute HR check-in**:

- [ ] Verify benefits enrollment completed

- [ ] Collect feedback on onboarding experience

## Troubleshooting

### Common Issues

**Issue**: I-9 documentation not acceptable
**Solution**: Provide list of acceptable documents, reschedule if needed. Cannot start work without valid I-9.

**Issue**: Benefits enrollment deadline missed
**Solution**: Special enrollment exception requires VP approval. Otherwise, must wait for open enrollment.

**Issue**: Compliance training not completed
**Solution**: System access may be limited. Required completion before Week 2.

**Issue**: First paycheck incorrect
**Solution**: Contact payroll immediately. Correction processed in next pay cycle.
